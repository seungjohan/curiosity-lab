# Triathlon Photo Finder

## Overview
An AI-powered tool to help athletes find their photos in large event galleries (triathlons, marathons) by recognizing personal gear and bib numbers.

## Key Features
- **Multi-Factor Recognition**:
  - Bib number (Participant number).
  - Face recognition.
  - Gear recognition: Swim cap, Bike brand/color, Cycle suit, Helmet, Goggles, Running shoes.
- **Fast Filtering**: Real-time or high-speed batch processing to find matches.

## Market Analysis
- **Competitor**: 'Our Symbol' (operating a similar service).
- **Opportunity**: Focus on specific sports like Triathlon which has very distinct gear across three disciplines.

## Challenges
- **Data Training**: High precision required for gear recognition.
- **Speed**: Processing large amounts of photos efficiently.
- **Scraping**: Leveraging internet product images to improve training data.
