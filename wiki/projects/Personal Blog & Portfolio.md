This project focuses on building a personal blog and portfolio website, leveraging AI tools and modern design principles. It serves as a central hub for Seungjo Han's identity, professional portfolio, and long-form writing.

## Prompt History Summary
The project evolved through 32+ iterations of prompts, focusing on:
1. **Initial Prototyping**: Setting up the basic blog/portfolio structure.
2. **Feature Expansion**: Adding hashtag filtering, "About" page, and responsive layouts.
3. **Refinement**: Polishing the footer, header, and blog post layout (Substack-inspired).
4. **Project Case Studies**: Redesigning project cards to show Impact/What I Did/Outcome.
5. **Final Polish**: Reverting Home to a YC-inspired minimal style and optimizing for mobile.
